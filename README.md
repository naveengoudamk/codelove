# CodeLove - Master the Code

**CodeLove** is a premium, modern coding practice platform designed to help developers master data structures and algorithms. Built with **Next.js 16**, **TypeScript**, and **CSS Modules**, it offers a high-performance, aesthetically pleasing environment for coding interviews preparation.

![CodeLove Preview](https://placehold.co/1200x600/171717/ededed?text=CodeLove+Preview)

## üöÄ Features

-   **Modern UI/UX**: A sleek, dark-themed interface with glassmorphism effects and radiant gradients.
-   **Integrated IDE**: Full-featured **Monaco Editor** (VS Code's editor) for a familiar coding experience.
-   **Problem Library**: A curated list of popular algorithmic problems (Two Sum, Median of Two Sorted Arrays, etc.).
-   **Split-View Workspace**: Resizable coding workspace with problem descriptions and code editor side-by-side.
-   **Responsive Design**: Fully responsive layout that works seamless on desktop and tablet.
